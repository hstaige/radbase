radbase documentation
=====================

As described in the recent white paper, the nuclear radius community requires modern tools for the compilation and evaluation
of nuclear charge radius data. radbase allows for a generalized least squares analysis of correlated data points, and attempts
to identify correlated data in the conversion process.
